Lesson Learn

Business Process

Proses bisnis adalah suatu kumpulan aktivitas atau pekerjaan terstruktur yang saling terkait untuk menyelesaikan suatu masalah tertentu atau yang menghasilkan produk atau layanan (demi meraih tujuan tertentu). Suatu proses bisnis dapat dipecah menjadi beberapa subproses yang masing-masing memiliki atribut sendiri tetapi juga berkontribusi untuk mencapai tujuan dari superprosesnya.

DFD (Data Flow Diagram)

Data Flow Diagram (DFD) adalah diagram yang menggambarkan aliran data dari sebuah proses atau sistem informasi. Pada DFD, terdapat informasi terkait input dan output dari setiap proses tersebut. Penggunaan DFD banyak digunakan untuk membantu para pengembang aplikasi, khususnya dalam proses pembuatan sebuah sistem informasi. DFD ini pertama kali dipopulerkan oleh Larry Constantine dan Ed Yourdon pada tahun 1970.

ERD (Entity Relationship Diagram)

ERD (Entity Relationship Diagram) atau diagram hubungan entitas adalah diagram yang digunakan untuk perancangan suatu database dan menunjukan relasi antar objek atau entitas beserta atribut-atributnya secara detail.

HTML (HyperText Markup Language)

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan mendesain halaman web. HTML memberikan struktur dasar pada halaman web dengan mendefinisikan elemen-elemen seperti judul, paragraf, tautan, gambar, dan konten lainnya. HTML bekerja bersama CSS (untuk tata letak dan tampilan) dan JavaScript (untuk interaktivitas) untuk menghasilkan sebuah situs web yang fungsional dan menarik secara visual.

Cascading Style Sheets (CSS)

Cascading Style Sheets (CSS) adalah bahasa pemrograman yang digunakan untuk mengatur tampilan dan tata letak halaman web. Dengan CSS, kamu bisa mengubah warna teks, jarak antar elemen, ukuran kolom, latar belakang, dan banyak lagi. Pada dasarnya, CSS membuat situs web terlihat menarik dan berfungsi dengan baik dengan memisahkan konten dari desain.
